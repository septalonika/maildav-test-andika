### How to configure

Requirement:

1. install UV
2. install Python
3. install Python 3.11

```
    uv init
    uv venv
    uv python --version
    uv python install 3.11.11
    uv python pin 3.11.11
    uv pip install flask
    uv add requests
    uv add flask
    flask --app main run --port 2500 --reload --debug
```
